# 분류 규칙 상세

## 규칙 우선순위 (위에서부터 먼저 적용)

### R1: Notion 팀활동비 매칭
- **조건**: Notion 팀활동비 DB에서 (날짜, 금액)이 일치하는 항목 존재
- **사용내역**: "팀 커피"
- **계정과목**: 복리후생비[회식비]
- **경비금액**: 전액
- **동반자**: Notion에서 조회된 동반자 목록

### R2: 택시 가승인 취소
- **조건**: 가맹점명 == "카카오T택시_가승인"
- **사용내역**: "택시 카드 선승인 취소 건"
- **계정과목**: 여비교통비[택시]
- **경비금액**: 0원

### R3: 법인택시 (야근)
- **조건**: 가맹점명이 "카카오T일반택시(법인)"으로 시작
- **사용내역**: "야근 택시비(배포)"
- **계정과목**: 여비교통비[택시]
- **경비금액**: 전액

### R4: 일반 택시
- **조건**: 가맹점명에 "택시" 포함
- **계정과목**: 여비교통비[택시]
- **경비금액**: 전액
- **경로**: 본사(강남) -> 집(김포)
- **수기 필드**: 사용내역 (수기 입력 필요)

### R5: 카페 (Notion 미매칭)
- **조건**: 가맹점명에 카페 키워드 포함 (Notion 매칭 안 됨)
- **카페 키워드**: 스타벅스, 투썸, 바나프레소, 컴포즈, 커피, 카페, CAFE, 로스터리, 이디야, 할리스, 메가커피, 빽다방, 커핀그루나루, 드롭탑, 파스쿠찌, 엔제리너스, 탐앤탐스
- **사용내역**: "팀 커피(확인필요)"
- **계정과목**: 복리후생비[회식비]
- **경비금액**: 전액
- **시간대 무관**: 카페 키워드가 감지되면 시간에 관계없이 적용

### R6: 점심 식대
- **조건**: 11:00~13:59 (카페/PG 제외)
- **사용내역**: "점심 식대" (PG 가맹점이면 "점심 식대(확인필요)")
- **계정과목**: 복리후생비[식비]
- **경비금액**: min(결제금액, 10,000원)
- **동반자**: 양경희 (금액 ≥ 20,000원이면 "양경희,확인필요")
- **PG 가맹점**: 네이버페이 등 실제 가맹점을 알 수 없는 결제수단

### R7: 저녁 식대
- **조건**: 17:00~20:29 (카페/PG 제외)
- **사용내역**: "저녁 식대" (PG 가맹점이면 "저녁 식대(확인필요)")
- **계정과목**: 복리후생비[식비]
- **경비금액**: min(결제금액, 12,000원)
- **동반자**: 양경희 (금액 ≥ 24,000원이면 "양경희,확인필요")

### R8: 수기
- **조건**: 위 규칙에 해당하지 않는 모든 건
- **모든 필드 수기 입력 필요**: 사용내역, 경비금액, 계정과목, 동반자
- **해당 시간대**: 00:00~10:59, 14:00~16:59, 20:30~23:59 (카페/택시 아닌 경우)

## "(확인필요)" 태그 의미

사용내역에 "(확인필요)"가 포함된 항목은 자동 분류는 되었지만 사용자 검토가 필요한 건입니다:
- **팀 커피(확인필요)**: Notion 팀활동비에 등록되지 않은 카페 결제. 실제 팀활동비인지 확인 필요
- **점심/저녁 식대(확인필요)**: 네이버페이 등 실제 가맹점을 알 수 없는 결제. 실제 식비인지 확인 필요
- **양경희,확인필요** (동반자): 식대 한도의 2배 이상 금액. 동반 식사인지 확인 필요
